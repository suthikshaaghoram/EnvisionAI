# EnvisionAI Frontend

This is the frontend for EnvisionAI, a React-based application built with Vite and Shadcn UI.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features

- **Manifestation Generator**: Interactive form to input personal details and generate personalized manifestations.
- **Audio Playback**: Listen to the generated manifestation using TTS.
- **Beautiful UI**: Designed with a calm and engaging aesthetic.
